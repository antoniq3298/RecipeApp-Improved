# RecipeApp-Improved
Java console app for managing recipes with SQLite.

## Features
✅ Add, list recipes  
✅ Uses SQLite DB  
✅ Clean layered structure

## How to run
```
javac -cp .:sqlite-jdbc.jar src/*.java
java -cp .:sqlite-jdbc.jar Main
```
